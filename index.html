📘 設定手順（初回のみ）
  1. Google Cloud Console でプロジェクトを作成
  2. Google Drive API を有効化
  3. OAuth同意画面を「外部」で作成（スコープ: drive.readonly）
  4. 「OAuth クライアントID」を作成（種類: ウェブアプリ）
  5. 承認済みJavaScript生成元にこのダッシュボードのURLを登録
  6. 発行されたClient IDを上の欄に貼り付け → 保存
